## 本地开发环境约定

- 前端开发统一使用 `Node 20 LTS`。
- 推荐用 `fnm` 管理多版本 Node，避免和其他老项目的 Node 16 冲突。
- 一键启动脚本 [scripts/start-all.bat](D:\java\project\docflow-ai\scripts\start-all.bat) 会优先使用 `fnm` 切到 Node 20，并在首次切换后重新安装前端依赖。
- 后端继续使用 `JDK 17`，启动脚本内部已经按项目级别设置 `JAVA_HOME`，不要求你手改全局环境变量。
